# path-exists-cli

> Check if a path exists, [cross-platform](https://github.com/sindresorhus/path-exists-cli/issues/1).

## Install

```
$ npm install --global path-exists-cli
```

## Usage

```
$ path-exists --help

  Usage
    $ path-exists <path>

  Example
    $ path-exists foo && echo "exists" || echo "doesn't exist"
```
